2. In General

This website uses no cookies, no tracking, no analytics, statistics or advertising services, and embeds no external fonts, maps, videos or social media plugins. Only the technically necessary data required to operate and secure the site is processed.

3. Hosting and Server Log Files

This website is hosted by hosting.de GmbH, Franzstr. 51, 52064 Aachen, Germany.

When you access this site, the web server automatically collects information that your browser transmits (so-called server log files):

  • IP address of the requesting device
  • Date and time of access
  • Name and URL of the file retrieved
  • Amount of data transferred
  • Referrer URL (the previously visited page)
  • Browser and operating system used

This data is technically required to correctly deliver the website and to ensure its stability and security. The legal basis is Art. 6(1)(f) GDPR (legitimate interest in reliable and secure operation). Log files are automatically deleted after 14 days at the latest.

4. Download and Download Counter

When downloading Lumora, the setup file is delivered and an anonymous total counter is incremented by one. No personal data – in particular no IP addresses – is stored in the process; the counter contains only a running total of downloads. The download itself appears, like any page request, in the server log files mentioned under point 3.

5. The Lumora Application (Installed Software)

In addition to this website, the installed Lumora application itself processes data when you actively use certain features:

  • Cover and artwork search: When you add a game or search for a cover, Lumora sends the game title to the search interfaces of Valve (Steam) and Microsoft (Microsoft Store), and – only if you have entered your own API key – to SteamGridDB, in order to find matching artwork. This inherently transmits your IP address to these providers; there is no sign-in, no profiling, and no further personal data is shared.
  • CPU sensors (optional): If CPU temperature/power display is needed, Lumora downloads the signed PawnIO driver from its official source (pawnio.eu) when required.
  • Streaming feature: When you live-share your screen, video and audio are transmitted directly (peer-to-peer, WebRTC) between you and the viewers you invite – with no server in between. Establishing the connection and shared links involves a brief exchange of data via our own relay infrastructure (lumora-streaming.de, see point 3) or, if you configure it in settings, via a self-hosted server of your choice.

This processing takes place exclusively on your own device or through requests you actively trigger; Lumora itself does not perform any tracking or analysis of your usage behaviour. The legal basis here, too, is Art. 6(1)(f) GDPR (legitimate interest in providing the feature you are using).

6. SSL/TLS Encryption

For security reasons, this site uses SSL/TLS encryption. You can recognize an encrypted connection by "https://" in the address bar and the lock icon in your browser.

7. External Links

This website contains links to external third-party websites (e.g. GitHub). We have no influence over their data processing; the respective provider's own privacy policy applies.
